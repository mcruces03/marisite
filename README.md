# Casa Uma — Next.js rebuild

Static-friendly Next.js 15 (App Router) site for Casa Uma (Spanish-language psychotherapy practice).

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

## Content

- Blog posts live in `content/posts/*.mdx`.
- Contact form opens a prefilled `mailto:` link (no backend, no env vars).
